# Getting Started:

1. Install Python Dependency Packages:
mac:
run `pip3 install -r requirements.txt`


windows:
`pip install -r requirements.txt`

2. Run the Server!
``` bash
$ export FLASK_APP=app.py
$ export FLASK_ENV=development
$ flask run
```
